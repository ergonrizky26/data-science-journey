# Data Preprocessing

## Machine Learning Process dibagi atas : 
1. Supervised Learning
2. Unsupervised Learning

## Supervised Learning 
**Supervised Learning** metode machine learning **dimana model belajar dari data yang sudah memiliki label**
  
  - Ada input (fitur) dan output (label)
  - Digunakan untuk mempelajari hubungan input dan output **Prediksi**

## Unsupervised Learning
**Unsupervised Learning** metode machine learning **dimana data tidak memiliki label**
Model berusaha menemukan pola, struktur dalam data tanpa diberi tahu mana output yang benar.

- Hanya ada input, tidak ada output
- Digunakan untuk menemukan pola dalam data
